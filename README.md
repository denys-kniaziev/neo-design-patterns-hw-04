# Woolf University. Design Patterns Course. Homework – Creational Patterns

Practice creational design patterns in TypeScript.

## Topic

Implement three patterns:

* Singleton
* Builder
* Prototype

## Main Logic

The app should demonstrate:

* Creating a single shared configuration instance
* Building structured documents step by step
* Cloning objects as independent deep copies
* Hiding and controlling object creation
* Verifying each pattern in a separate `main.ts`

## Run

```bash
npx ts-node src/singleton/main.ts
npx ts-node src/builder/main.ts
npx ts-node src/prototype/main.ts
```
